🔑 기능 설명

### 1) 회원가입/로그인

- JWT인증 방식으로 로그인 구현
- ID 중복확인, 각 필드별 유효성 체크

### 2) 게시글 CRUD

- 게시글 목록 전체 조회, 상세조회, 등록, 수정, 삭제
- 이미지 파일 업로드

### 3) 댓글 CRUD

- 댓글 등록, 조회, 수정, 삭제

### 4) 좋아요 기능

- 좋아요 버튼 클릭시 좋아요 숫자가 증가하도록 구현

### 5) 검색 기능

- 제목, 본문에 검색어가 포함되면 검색되도록
